# Hawai-i-Tourism-Length-of-Stay-Analysis-Project

This project analyzes how visitor behavior in Hawai‘i has evolved over the past two decades, with a focus on length of stay (LOS). Using historical data from 1999–2021, the dashboard highlights long‑term trends, differences between first‑time and repeat visitors, island‑level patterns, and the impact of the COVID‑19 disruption in 2020–2021. The goal is to understand which islands attract longer stays, how visitor patterns have shifted, and where the most significant year‑over‑year changes occurred.

Kaggle Dataset used: https://www.kaggle.com/datasets/csafrit2/hawaii-travel-length-of-trip/data

Key Questions
- How has the average length of stay changed over time?
- Do first‑time and repeat visitors behave differently?
- Which islands have the longest and shortest trip durations?
- How did COVID‑19 affect visitor behavior in 2020 and 2021?
- Where were the largest year‑over‑year changes in LOS?

Tools Used: SQL, Tableau

Live Tableau Dashboard: https://public.tableau.com/app/profile/daniel.naqvi/viz/HawaiiTourismLengthofStayAnalysis/Dashboard1?publish=yes
